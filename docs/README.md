# Documentation Directory

This folder contains all documentation files for the Cricket Player Auction application.

## Configuration Guides

- **CONFIGURATION_GUIDE.md** - Complete guide to configuring the auction system (teams, grades, quotas, purse)
- **EXCEL_COLUMN_CONFIG.md** - Guide for configuring Excel column mappings for player import
- **HOW_TO_CONFIGURE_AUCTION.txt** - Quick reference for auction configuration
- **PLAYER_STATS_GUIDE.md** - Guide for adding player statistics to the system

## Deployment Documentation

- **AZURE_DEPLOYMENT.md** - Complete guide for deploying to Microsoft Azure
- **QUICK_START_AZURE.md** - Quick start guide for Azure deployment
- **DEPLOYMENT_CHECKLIST.md** - Pre-deployment checklist

## Technical Documentation

- **replit.md** - Main project documentation, system architecture, and recent changes
- **design_guidelines.md** - UI/UX design guidelines and component usage
- **PACKAGE_JSON_UPDATES.md** - Package updates and dependency information

## Quick Links

### To Change League Name
1. Open `client/src/config/leagueConfig.ts`
2. Edit the `LEAGUE_NAME` constant
3. Save the file

### To Configure Auction Settings
See `CONFIGURATION_GUIDE.md` for complete instructions on:
- Adding/removing teams
- Setting grade quotas
- Configuring base prices
- Managing team logos

### To Import Players
See `EXCEL_COLUMN_CONFIG.md` for Excel import setup and column mapping.
