==================================================================

To customize the auction settings for each new auction, you only need to 
edit ONE file: client/public/config.json

No code changes required!

------------------------------------------------------------------
STEPS TO CONFIGURE FOR A NEW AUCTION
------------------------------------------------------------------

1. Open: client/public/config.json

2. Update the values as needed:
   - Change base prices
   - Change increments
   - Add/remove/modify teams
   - Update team purse amounts
   - Adjust grade quotas

3. Save the file

4. Refresh your browser - changes will take effect immediately!

------------------------------------------------------------------
PLAYER DATA (EXCEL FILE)
------------------------------------------------------------------

Location: client/public/players.xlsx

The Excel file should contain:
- Column: name     (e.g., "Virat Kohli")
- Column: grade    (e.g., "A", "B", or "C")
- Column: photo    (e.g., "Virat_Kohli.png")

Player images should be in: client/public/player_images/
